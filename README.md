# Elden Ring Character Gallery
**CSC340 - Static Web App Assignment**

A responsive, multi-page static web application showcasing legendary characters from the *Elden Ring* universe. Built using HTML, CSS, and the Bootstrap 5 framework.

## Features
* **Character Gallery**: A responsive grid featuring characters of Elden Ring
* **Live Search**: Real-time filtering functionality allowing users to search by name or role.
* **Lore Details**: Deep-dive pages for individual characters including backstories and abilities.
* **Golden Order UI**: A custom-styled dark theme with gold accents, mimicking the aesthetic of the game.
* **Prototype Form**: An interface for adding new legends to the database.

## Project Structure
* `index.html`: The main landing page and character gallery.
* `details.html`: Detailed lore page for Queen Marika.
* `about.html`: Information regarding the project's purpose and theme choice.
* `new-character-form.html`: Prototype submission form for new characters.
* `/images`: Assets including character portraits and landscape backgrounds.

## Technologies Used
* **HTML5 & CSS3**: Core structure and custom styling.
* **Bootstrap 5**: Responsive grid system and UI components.
* **JavaScript**: Logic for the gallery search filter.
* **GitHub Pages**: Web hosting and deployment.
