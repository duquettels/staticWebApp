About This Gallery

This static web application was designed and developed as a project for CSC340. The primary goal was to practice building responsive, multi-page prototypes using HTML, CSS, and the Bootstrap framework.

Why Elden Ring?

I chose Elden Ring as the theme for this gallery due to its awesome character design and the complex narrative crafted by FromSoftware and George R.R. Martin. Each character featured in this gallery plays a vital role in the history of the Lands Between.

Technical Features

  • Responsive Grid Layout using Bootstrap.
  • Live Search functionality to filter character legends.
  • Uniform image scaling with CSS object-fit properties.
  • Consistent site-wide navigation and branding.
